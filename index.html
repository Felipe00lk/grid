HTML, CSS e JAVASCRIPT

O HTML (HipperText Markup Language, ou Linguagem de Marcação de Hipertexto) é o componente básico da web. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação que permite inserir conteúdo e estabelecer a estrutura básica de um website. Usando um conjunto de "etiquetas" (tags), o HTML organiza e define o significado e a estrutura do conteúdo, como títulos, parágrafos, links e imagens, para que os navegadores possam renderizar a página corretamente.
O CSS (Cascading Style Sheets) é uma linguagem fundamental para o desenvolvimento web, utilizada para estilizar e controlar a apresentação visual de documentos HTML. Ele atua separando o conteúdo (definido pelo HTML) do design, permitindo que você gerencie cores, fontes, layouts, espaçamentos e a responsividade de um site (como ele se adapta a diferentes tamanhos de tela), proporcionando uma experiência de usuário visualmente atraente e consistente.
JavaScript (JS) é uma linguagem de programação usada para adicionar interatividade e comportamento dinâmico a sites. É fundamental para a web e permite criar elementos como animações, menus, validação de formulários e atualização de conteúdo em tempo real, sem que o usuário precise recarregar a página. Além do desenvolvimento web front-end (no navegador), o JS também pode ser usado no back-end (lado do servidor) com ferramentas como o Node.js, e para criar aplicativos móveis, jogos e mais.
